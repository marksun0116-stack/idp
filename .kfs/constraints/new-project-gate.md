# New project gate (RULE-013)

Before first driving **FEAT-*** or net-new **`src/`** for a product:

- **`docs/prd/`** — ≥1 PRD (`*.md`)
- **`docs/architecture/`** — ≥1 architecture intent doc

**Greenfield:** create from stated goals (agent may draft).

**Existing application (most adoptions):** run **[reverse-engineer-baseline.md](../agent/runbooks/reverse-engineer-baseline.md)** first — baseline **`docs/`** + **`knowledge/`** from **`src/`**, **`tests/`**, and optional **`references/`**. No separate current/target doc trees required.
